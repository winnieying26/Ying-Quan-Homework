inlinelevel: <span>, <img>, <a>

start on new line
Take full width available

emphasized: <strong>, <em>

table:
<table>
<thead> Groups the header content of a table
<tr> (Table Row):
Defines a single row within the table.
<th> (Table Header):
Creates a header cell, typically bold and centered by default.
<tbody>:
Contains the main body of the table (the data).
<td> (Table Data):
Defines a single cell in a table row.
<tfoot>:
Groups the footer content of a table (often used for summaries).
