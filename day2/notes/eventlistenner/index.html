Event Listener

The once option is a Boolean option in the addEventListener() method. If set to true, it ensures that the event listener is invoked at most once after being added. Once the event listener is triggered, it will automatically be removed. How it works: When an event listener is added with the once: true option, it will be triggered the first time the event occurs, and then it will be removed from the element after being executed, ensuring it only runs once.
